Telegram bot ulash

1-qadam — Yangi bot yarating
  1. Telegram'da @BotFather ni oching.
  2. Yangi suhbat boshlab, /newbot deb yozing.
  3. Botingizning nomini bering (masalan: Aros Tasks).
  4. Username bering — oxiri bot bilan tugashi kerak (masalan: arostasks_bot).
  5. BotFather sizga tokenni qaytaradi — uni nusxalang, keyingi qadamga keling.
💡 Token shunday ko'rinishda bo'ladi: 7123456789:AAH...XYZ. Uni hech kim bilan ulashmang — bu botingizning paroli.
2-qadam — Tokenni kiriting

BotFather bergan tokenni shu yerga joylashtiring. Biz uni Telegram orqali tekshirib, bot ma'lumotlarini olamiz.

3-qadam — Tasdiqlang
Saqlangach, biz botingizga webhook o'rnatamiz. Shundan keyin xodimlar botga /start bosib hisoblarini bog'lay oladi.

Webhook URL

Vazifa hodisalari (yaratildi, holat o'zgardi, biriktirildi) sizning n8n instance'ingizdagi webhook'gа JSON sifatida yuboriladi.

💡 n8n'da yangi workflow yarating → "Webhook" trigger qo'shing → URL'ni nusxalab shu yerga joylang. Misol payload:
{ "event": "task.created", "workspace_id": "...", "task": { ... } }
Yuboriladigan hodisalar:
  • task.created — yangi vazifa yaratilganda
  • task.status_changed — holat o'zgarganda
